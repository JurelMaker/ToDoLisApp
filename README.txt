Ejercicio: Crear una ToDoList App
Objetivo: Crear una aplicación de lista de tareas con Flask (o el framework de tu preferencia) que permita a los usuarios 
agregar tareas, cambiar el estado, establecer fechas de 
finalización, organizar las tareas en categorías, y filtrarlas por categoría.

Desarrolla una aplicación de lista de tareas (ToDoList App) que cumpla con los siguientes requisitos:

Permita agregar nuevas tareas con:

Título
Descripción
Estado (pendiente o completa)
Fecha de finalización
Categoría (ej., "Trabajo", "Personal", "Estudios", etc.)
Permita ver una lista de todas las tareas, mostrando:

Título
Estado
Fecha de finalización
Categoría
Permita cambiar el estado de las tareas entre "Pendiente" y "Completa".

Permita filtrar las tareas por categoría para mostrar solo las de una categoría específica.
